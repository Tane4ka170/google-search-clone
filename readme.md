# Google Clone

A minimal front-end clone of Google Search built with HTML, CSS, and JavaScript.

## Features

- Clean Google-like UI
- Search input with dynamic query redirect
- Responsive layout

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How It Works

- User enters a query
- Click triggers a redirect to:
  https://www.google.com/search?q=<query>

## Run Locally

1. Clone or download the project
2. Open `index.html` in a browser

## Notes

- No backend
- No API usage
- Pure client-side behavior
